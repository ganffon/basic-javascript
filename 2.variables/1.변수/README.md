# 01. 변수

자바스크립트에서는 변수를 사용하여 여러가지 형태의 데이터를 저장할 수 있습니다.

이번 시간에는 `let` 키워드 사용하여 변수를 선언해보도록 하겠습니다.

변수는 아래와 같이 `let` 키워드를 앞에 붙여서 선언할 수 있습니다.

```js
let ourName = 'wecode';
```

`let` 키워드를 사용하여 `ourName` 변수를 선언하고,

그 값으로 "wecode" 라는 문자열 타입의 값을 할당해주었습니다.
