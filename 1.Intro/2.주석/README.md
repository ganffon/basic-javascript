# 01. 주석 처리하기

코드에서 주석은 코드를 설명하는 코멘트로 실제 동작하는 코드가 아닙니다. 한 줄을 주석처리하는 경우에는 slash 두개( `//` )로 시작합니다.

```js
// (코드 한줄)
```

여러 줄을 주석처리하는 경우에는 아래와 같이 감싸주면 됩니다.

```js
/* 
  (코드 여러줄) 
*/
```
