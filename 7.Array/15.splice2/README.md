# 14. splice 2

## Assignment

기념사진을 찍으려고 하는데, 키 큰 사람들이 앞으로 나와있습니다.

splice method를 이용해 175cm 이상인 사람은 뒤에 설 수 있도록 코드를 작성해주세요.

(단, splice 메서드 괄호 안에는 무조건 음수만 적용해야 합니다)

```js
let lane1 = [
  { 철수: '150cm' },
  { 영희: '153cm' },
  { 바둑이: '155cm' },
  { 밍키: '160cm' },
  { 살구: '168cm' },
  { 두팔: '175cm' },
  { 여름: '180cm' },
  { 소헌: '181cm' },
];

extractOver175(lane1); // [{'두팔': '175cm'}, {'여름': '180cm'}, {'소헌': '181cm'}];
```
