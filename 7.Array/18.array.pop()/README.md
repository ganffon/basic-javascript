# 2. Array.pop( )

변수를 조작하는 또 다른 방법 중의 하나는 .pop() 메소드를 사용하는 것입니다.

`.pop()` 메소드는 배열의 마지막 요소를 제거하며 제거된 요소를 반환합니다.

```js
let threeArr = [1, 4, 6];
let oneDown = threeArr.pop();

console.log(oneDown); // Returns 6
console.log(threeArr); // Returns [1, 4]
```
